manjaro装机后基本配置

May 21, 2020
1 min read

  • 添加archlinuxcn的源

    1. sudo nano /etc/pacman.conf

    2. 在文件末尾输入： [archlinuxcn]

    SigLevel = Optional TrustedOnly

    Server = http://mirrors.163.com/archlinux-cn/$arch 3.按ctrl+x退出，输入y保存

  • 更新与同步 sudo pacman -Syyu

    sudo pacman -S archlinuxcn-keyring

    sudo pacman -Syyu

  • 安装输入法

    1. 为了方便所以先安装yay ：sudo pacman -S yay

    2. 安装fcitx : yay -S fcitx-im kcm-fcitx fcitx-sogouimebs

    3. 在~目录下新建.xprofile文件：sudo nano ~/.xprofile

    4. 输入内容 export GTK_IM_MODULE=fcitx export QT_IM_MODULE=fcitx export XMODIFIERS=”@im=fcitx"
